1. Устанавливаем глобально vue из https://cli.vuejs.org/guide/installation.html:
	npm install -g @vue/cli
	это делается один раз на компе разработчика

2. Создаем новый проект и добавляем туда vue
	vue create my-app

3. Install vue tooling for VS Code (там в т.ч. подсветка)

4. Запуск проекта для теста
	npm run serve

5. Ставим роутер локально (нужен для многостраниченых приложений)
	vue add router

6. Ставим vuex (нужен для ведения общего хранилища)
	vue add vuex
	https://vuex.vuejs.org/ (всё тут о vuex)

7. Ставим vuetify (готовые компоненты со стилями)
	vue add vuetify
